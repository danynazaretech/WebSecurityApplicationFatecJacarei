Minha Área de Códigos

Primeiro verifique se o Node e o npm node -v && npm -vestá instalado, e se verdadeiro inicialize o Node em seu projeto ;npm init -y. E em seguida instale o typescript para poder compilar códigos no seu projeto e o @types/node para integrar o TypeScript com o Node. Para analisar a versão digite tsc -ve então estando instalado o compilador inicie o projeto com npx tsc --init.

Iniciando Projeto TypeScript
Copiado! 
npm init - y
npm install typescript  @types/node  -D
npx tsc --init

Para configurar corretamente o tsconfig.json de acordo com a versão do Node.js que você está utilizando, é essencial alinhar as opções de target, lib e module com os recursos de ECMAScript suportados por essa versão. A Microsoft mantém uma página no GitHub , que fornece recomendações específicas para diferentes versões do Node.js. Para isso execute as seguintes instruções:

  1. verifique a versão do Node.js através do node -v;
  2. vá até o tsconfig.json e altere as configurações target, lib e module conforme especificado no documento;
  3. e para alinhar alinhar a pasta onde tem o arquivo .ts que vai ser compilado adicione a pasta mkdir src e em seguida altere a opção "rootDir": "./src".

Para adicionar o código do frontend dos laboratórios adicione a pasta mkdir public na raíz do projeto.

Para converter o código de TypeScript pra Java Script de forma automatizada adicione a biblioteca npm install tsx -D. E logo em seguida configure "scripts" do package.json para que possa reiniciar o servidor automaticamente.

Configure Script.js
Copiado! 
"scripts": {
     "dev":"tsx watch src/server.ts"
},

Para testar o funcionamento do servidor primeiro adicione o arquivo src/server.ts, e o execute npm run dev

CMD
Copiado! 
echo "console.log(\"Olá mundo\");" > src/server.ts

execute npm run dev
